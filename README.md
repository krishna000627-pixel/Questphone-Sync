# QuestPhone Sync Backend
Git-based sync for QuestPhone. No server needed.

## Files
- data/user.json — user profile, level, coins, streak
- data/quests.json — all quests

## Sync from app
Push/pull via WiFi Sync screen in the app.
Or use GitHub API: GET/PUT https://api.github.com/repos/USER/REPO/contents/data/user.json
